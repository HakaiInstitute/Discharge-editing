Download current data from the Hakai Institute sensor network portal (.csv) and rename files:
'Qrate_5min.csv' --> 5 minute discharge (m3/s) data from all watersheds
'Qrate_hourly.csv' --> hourly discharge (m3/s) data from all watersheds
'Qrate_daily.csv' --> daily discharge (m3/s) data from all watersheds
'Qvol_5min.csv' --> 5 minute total discharge (m3) data from all watersheds
'Qvol_hourly.csv' --> hourly total discharge (m3) data from all watersheds
'Qvol_daily.csv' --> daily total discharge (m3) data from all watersheds

Edit the downloaded files to resemble the headers from the example files.
